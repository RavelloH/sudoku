# RavelloH's Sudoku

一个现代化的数独游戏应用，基于 Next.js 15 构建，提供完整的数独游戏体验和智能功能。

## ✨ 特性

### 🎮 多种游戏模式
- **挑战模式** - 计时挑战，包含智能提示和无限撤销/重做
- **批量生成** - 生成多个数独题目，支持打印和导出
- **自动解题** - 智能解题器，实时推导和显示解题过程
- **历史记录** - 完整的游戏历史管理和回放功能

### 🧠 智能功能
- **6个难度级别** - 简单、中等、困难、专家、大师、极限
- **智能计时器** - 只在第一次输入数字时开始计时
- **智能提示系统** - 不仅高亮，还能填入数字并说明原因
- **实时冲突检测** - 自动检测并高亮显示冲突数字
- **自动保存** - 游戏进度自动保存到本地存储

### 🎨 用户体验
- **响应式设计** - 完美适配桌面和移动设备
- **深色/浅色主题** - 支持系统主题自动切换
- **移动端优化** - 数字轮盘输入，长按操作
- **键盘支持** - 完整的键盘快捷键操作
- **动画效果** - 流畅的交互动画和完成庆祝效果

### ☁️ 数据管理
- **本地存储** - 游戏记录、统计数据、设置配置
- **云同步** - 支持云端存档和跨设备同步
- **数据导入/导出** - 完整的游戏数据备份功能
- **统计分析** - 详细的游戏统计和最佳时间记录

## 🛠️ 技术栈

- **框架**: Next.js 15.4.6 + React 19.1.0 + TypeScript 5
- **样式**: Tailwind CSS 4 + shadcn/ui
- **动画**: Framer Motion 12.23.12
- **图标**: Lucide React 0.540.0
- **主题**: next-themes 0.4.6
- **包管理**: pnpm
- **开发工具**: ESLint 9, Turbopack

## 🚀 快速开始

### 环境要求
- Node.js 18+ 
- pnpm 8+

### 安装和运行

```bash
# 克隆项目
git clone https://github.com/ravelloh/sudoku.git
cd sudoku

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 打开浏览器访问 http://localhost:3000
```

### 构建和部署

```bash
# 创建生产构建
pnpm build

# 运行生产服务器
pnpm start

# 代码检查
pnpm lint
```

## 🎯 核心功能说明

### 挑战模式
- **智能计时**: 只有输入第一个数字时才开始计时
- **无限撤销/重做**: 完整的操作历史记录
- **智能提示**: 分析当前局面，提供具体的解题建议
- **冲突检测**: 实时检测并高亮显示违反数独规则的数字
- **自动保存**: 游戏进度自动保存，刷新页面不会丢失

### 批量生成模式
- **多题目生成**: 一次生成1-50个数独题目
- **难度选择**: 支持所有6个难度级别
- **打印优化**: 专为打印设计的布局，支持多种页面格式
- **答案显示**: 可选择显示题目或答案
- **导出功能**: 支持复制到剪贴板和直接打印

### 自动解题模式
- **智能求解**: 使用回溯算法快速求解任意数独
- **实时推导**: 逐步显示解题过程，帮助学习解题技巧
- **导入功能**: 可导入当前游戏状态进行分析
- **可视化**: 用不同颜色显示程序推导的数字

### 历史记录模式
- **完整历史**: 保存所有完成的游戏记录
- **统计信息**: 显示各难度的完成时间和成功率
- **回放功能**: 可以重新查看历史游戏
- **数据管理**: 支持删除记录和数据导出

## 🎮 操作说明

### 桌面端操作
- **数字键**: 填入数字
- **Delete/Backspace**: 清空格子
- **方向键**: 移动焦点
- **Z**: 撤销
- **Y**: 重做
- **H**: 提示
- **P**: 暂停/继续

### 移动端操作
- **点击格子**: 选中格子
- **数字轮盘**: 选择要填入的数字
- **长按数字**: 删除数字
- **双击**: 快速填入候选数字
